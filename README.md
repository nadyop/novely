# Novely

Toko buku online yang menjual novel-novel best seller.
Sebuah tugas akhir dari mata kuliah Sistem Terdistribusi 2017. Yang bersifat Restful API.



## Build with 

* CodeIgniter - Framework PHP

* Bootstrap - HTML Design

* RajaOngkir API

* Twitter API
